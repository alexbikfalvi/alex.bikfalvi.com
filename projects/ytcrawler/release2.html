YTCrawler Release 2

The YtCrawler software for crawling metadata from the video sharing site YouTube, is a project in progress and the software on this page is subject to change. There are no guarantees for this software and you can use it on your own risk. YTCrawler is free software and is licensed under the terms of GNU General Public License.

This page is a user guide to YtCrawler. Contact me to get the installation package and/or source code. A developer guide is not currently available.
