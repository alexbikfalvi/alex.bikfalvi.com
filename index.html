An Analytics Application for YouTube Measurements

YouTube is the world's most popular video sharing web site, and understanding its user activity can have a profound impact on designing new content distribution systems. YtAnalytics is a .NET application that allows you to browse the collection of YouTube videos, collect statistical data and trends using the built-in tools, and perform distributed measurements using the PlanetLab computing platform.
