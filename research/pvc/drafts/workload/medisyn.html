Video File Duration

The set of file durations is divided into groups, where the durations in each group have a normal distribution. The following figure shows the default distribution for file duration, with 4 groups each containing 63%, 10%, 18% and 9% of the files, respectively. The mean duration for each group are: 0, 2000, 3300 and 5821 seconds (the standard deviations are: 600, 400, 600 and 1223).

Session Prefix

The session prefix gives the fraction of file that is watched by a user during the session. Every file has an individual distribution for the session prefix that depends on the the duration of the file: shorter files have more "complete sessions", i.e. they are watched completely during a session. Without going into the details, the following figures illustrate the distribution of the session prefix for 3 different file, with durations 67, 241 and 3317 seconds.

New File Introduction

New files are being introduced in introduction days. The model of new file introduction has the following components:

a) The determination of the introduction days: this is done by assuming the first day is an introduction day, and modelling the gaps between the introduction days (these can follow a Pareto distribution - default, an exponential distribution, or constant). The following two figures shows a histogram of the gaps between introduction days using Pareto (most gaps between introduction days are of 1 day, then 2 days etc), and the CDF of file arrival where the first introduction day is day zero.

b) Calculating the number of new files that start being introduced with each introduction day. The distribution of these is Pareto:

c) For every introduction day, Medisyn calculates the time (during the day) when the first file for that introduction day is introduced. This follows a rotated normal distribution with introduction times in the evening and early morning for most introduction days.

d) When more that one file start being introducted within an introduction day, the first file is introduced at the hour determined using the distribution from (c), while for subsequent files, Medisyn calculates the gaps between the files. These gaps follow a Pareto distribution.

File Lifespan

Files are divided into two categories: news files and regular files. Files with a higher popularity have a higher probability to be news-like, whereas files with a lower popularity have a higher probability to be regular. The following two figures show a histogram of news and regular files, respectively, in 100-file bins, where the files are sorted by decreasing popularity.

News files have a Pareto shape of the lifespan, regular files have a lognormal shape of the lifespan, measured from their introduction day. The following two figures illustrate an example of file lifespan for news and regular file, respectively, on an absolute time axis (i.e. time starts from zero and file lifespan starts from their introduction time). The news file is the most popular file with 1700 sessions, the regular file is the third most popular with 1574 sessions.

Diurnal Access Pattern

Session times during a day have two components: (i) inter-session times during an hour have an exponential distribution, (ii) while during a day they follow a diurnal pattern with more sessions in the evening.
